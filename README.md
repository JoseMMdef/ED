El programa desarrollado consiste en la creación de un arreglo bidimensional en Python que permite registrar y gestionar las ventas mensuales de tres departamentos: ropa, deportes y juguetería. La matriz está organizada por filas que representan los doce meses del año y columnas que representan cada departamento. El objetivo principal del programa es simular un sistema básico de registro de ventas, permitiendo almacenar información de manera estructurada y facilitar su manipulación mediante diferentes operaciones dentro de un menú interactivo.

Para el funcionamiento del sistema se implementaron distintos métodos que permiten manipular la información dentro del arreglo. El método de inserción permite agregar o actualizar una venta en un mes y departamento específico, solicitando al usuario los datos necesarios para almacenarlos en la matriz. El método de búsqueda permite localizar y mostrar una venta registrada, facilitando la consulta de información almacenada. Asimismo, el método de eliminación permite borrar una venta específica, reemplazando el valor almacenado por cero, lo que permite corregir o eliminar registros cuando sea necesario.
